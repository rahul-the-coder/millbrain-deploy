# Draft a Maintenance Work Order

Generate a complete work order in the standard CMMS format. This is a Brain-reasoning task — no script to run. Use the context you have already gathered and fill in the template below.

## Prerequisites

Before drafting, confirm you have:
- [ ] `machineID`, `model`, `age` from the machines table
- [ ] Active error code and alarm timestamp
- [ ] Sensor readings and status from `check_sensors`
- [ ] Maintenance history and overdue status from `lookup_maint`
- [ ] Confidence score and component risk from the scoring logic
- [ ] Relevant SOP passages from `search_sop`

If any of these are missing, run the appropriate skill first.

## Reference Tables

**Error → Component mapping:**

| Error | Description | Primary Component | Component Description |
|-------|-------------|-------------------|-----------------------|
| error1 | Voltage Anomaly | comp1 | Motor / VFD (main drive) |
| error2 | Rotation Fault | comp2 | Bearing / Coupling assembly |
| error3 | Pressure Deviation | comp2 | Bearing / Coupling assembly |
| error4 | Vibration Excess | comp3 | Mechanical assembly (refiner plate/roll) |
| error5 | Composite Anomaly | comp4 | Mechanical seal |

**Model → Plant section:**

| Model | Location |
|-------|----------|
| model1 | Wet End |
| model2 | Dry End |
| model3 | Stock Prep |
| model4 | Utilities |

**Average replacement cycles:** comp1 = 61.1d · comp2 = 57.2d · comp3 = 62.0d · comp4 = 61.6d

**LOTO energy sources by model:**

| Model | Energy Sources |
|-------|---------------|
| model1 | Electrical |
| model2 | Electrical, Steam |
| model3 | Electrical, Mechanical (high-inertia rotating mass) |
| model4 | Electrical, Pneumatic |

## Priority Rules

Apply the first rule that matches:

| Priority | Criteria |
|----------|----------|
| **CRITICAL** | Confidence ≥ 80% AND (machine on critical path OR active failure confirmed) |
| **HIGH** | Confidence ≥ 80% OR sensor reading > 20% outside normal range |
| **MEDIUM** | Confidence 50–79% OR sensor trending toward threshold |
| **LOW** | Confidence < 50% OR single event with no supporting trend |

## Work Order Template

Fill in every `{placeholder}`. Do not leave any field as a placeholder — use "Unknown" or "To be confirmed" where information is genuinely unavailable. Leave Sign-Off and Completion Notes sections blank; annotate them as *(for human completion)*.

## After Drafting

State clearly: "This work order is a DRAFT. A qualified maintenance supervisor must review and approve before any work begins. LOTO must be applied before touching the equipment."

Do not suggest the operator proceed without supervisor approval.
